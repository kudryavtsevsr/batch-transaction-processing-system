# Batch Transaction Processing System

## Technologies Used
- React
- TypeScript
- Vite
- Zustand
- MUI
- Vitest

## Requirements
- Node.js v22 or higher

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Testing

```bash
# Run tests
npm run test
```
